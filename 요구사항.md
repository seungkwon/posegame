동작 순서 기억하기

화면에 여러 동작을 차례로 보여준 뒤, 참가자들이 같은 순서로 재현합니다.

예:

1단계: 양팔 들기
2단계: 오른손 들기 → 스쿼트
3단계: 오른손 들기 → 스쿼트 → 왼쪽 기울기
4단계: 오른손 들기 → 스쿼트 → 왼쪽 기울기 → 점프

이 게임은 참가자가 다음 정보를 잠시 저장하고 처리해야 합니다.

동작의 종류
동작의 순서
동작의 방향
동작의 개수
자신의 실행 결과

따라서 단순히 자세 하나를 따라 하는 것보다 작업기억을 직접적으로 사용합니다. 작업기억 훈련은 훈련한 과제와 유사한 기억 과제의 성능을 소폭 향상시킬 가능성이 있지만, 지능이나 일상생활 전반으로 효과가 크게 전이된다고 단정하기는 어렵습니다.

MediaPipe 판정 방식

각 동작을 클래스로 변환합니다.

0: 기본 자세
1: 양팔 들기
2: 왼팔 들기
3: 오른팔 들기
4: 스쿼트
5: 왼쪽 기울기
6: 오른쪽 기울기
7: 점프

정답:

target_sequence = [3, 4, 5, 7]

참가자 동작:

player_sequence = [3, 4, 5, 7]

순서가 모두 일치하면 성공입니다.


# 게임은 브라우저에서 할 수 있도록 합니다. 
# 프론트엔드에서는 로그인 없이 게임 스코어를 단계별로 제공합니다. 
# 포즈에 나오는 동작을 화면에 보여주고 따라하는 사용자도 보여주고 포즈도 일치하는지를 보여줍니다. 
# 애니메이션 효과를 이용해서 맞추었는지를 확실히 보여줍니다.
# 게임 난이도는 1단계에서 10단계까지 나눕니다.
# 미디어 파이프는 브라우저에서 추론을 하고 동작 일치 여부 순서 일치 여부까지 모두 브라우저에서 처리합니다.
