# Revision Log

## Step 1. 프로젝트 골격 및 기본 게임 흐름 구현

- 프론트엔드 React + Vite 구조 생성
- 백엔드 FastAPI 구조 생성
- PostgreSQL용 Docker Compose 추가
- 아이디 기반 로그인 API 및 화면 추가
- 단계별 시퀀스 생성 로직 추가
- 목표 시퀀스와 사용자 시퀀스 비교 로직 추가
- 게임 결과 저장 API 및 마이페이지 조회 화면 추가
- 포즈 엔진 연결 지점과 웹캠 UI 추가
- FPS 기준 문서를 계획서에 반영

## Step 2. MediaPipe 포즈 추론 및 동작 분류 구현

- 브라우저에서 MediaPipe PoseLandmarker 실시간 추론 연결
- 목표 FPS 12 기준의 추론 루프 구현
- 기본 자세, 양팔 들기, 왼팔 들기, 오른팔 들기, 스쿼트, 좌우 기울기, 점프 규칙 기반 분류기 추가
- 포즈 랜드마크 캔버스 오버레이 추가
- 실시간 감지 동작 표시 및 FPS 표시 추가
- 수동 테스트 버튼은 디버깅 보조 용도로 유지
- 프론트엔드 빌드 검증 완료

## Step 3. 라운드 프리뷰와 플레이 흐름 개선

- 목표 시퀀스를 먼저 보여주는 프리뷰 단계 추가
- 프리뷰 종료 후 3초 카운트다운 단계 추가
- 입력 단계 진행률 바와 완료 상태 표시 추가
- 현재 프리뷰 중인 동작을 카메라 영역 오버레이로 강조
- 단계별 상태 메시지와 흐름을 플레이 기준으로 정리

## 기록 원칙

- 각 구현 단계가 완료될 때마다 이 문서에 완료 내용과 변경 범위를 추가합니다.
